# hex-convert

```
支持 76 位以内的任意进制相互转换

Usage:
  hc [command]

Available Commands:
  b           转换为二进制
  d           转换为十进制
  h           转换为十六进制
  help        Help about any command
  o           转换为八进制
  server      启动 HTTP Server

Flags:
  -d, --data string   要转换数值
  -h, --help          help for hc
  -s, --scale int     要转换的进制
  -v, --version       show version of the hc.
```
